# Golam Robbanie - Portfolio Website

A clean, professional, multi-page portfolio website showcasing AI engineering experience, projects, and skills.

## üöÄ Quick Setup

### Adding Your Profile Picture

1. Take or choose a professional headshot photo
2. Rename it to `profile.jpg`
3. Place it in the root directory of this repository
4. Recommended size: 400x400px or larger (square format works best)
5. The image will automatically appear in a circular frame on the homepage

### Adding Your Resume

1. Export your resume as a PDF file
2. Rename it to `resume.pdf`
3. Place it in the root directory
4. The download button on the homepage will now work

### Customizing Favicon

The favicon (browser tab icon) is already created with your initials "GR". It will automatically appear in browser tabs.

## üì§ How to Deploy & Share

### Step 1: Merge to Main Branch

```bash
# Switch to main branch
git checkout main

# Merge your feature branch
git merge claude/setup-latex-document-01ECgoJyVxUCbKS7qA5mr1J9

# Push to GitHub
git push origin main
```

### Step 2: Enable GitHub Pages

1. Go to your repository on GitHub: `https://github.com/Golam-Robbanie-Sajib/Golam-Robbanie-Sajib.github.io`
2. Click on **Settings** (top right)
3. Scroll down to **Pages** in the left sidebar
4. Under **Source**, select **Deploy from a branch**
5. Select **main** branch and **/ (root)** folder
6. Click **Save**

### Step 3: Access Your Live Website

After 2-3 minutes, your site will be live at:
**https://golam-robbanie-sajib.github.io/**
